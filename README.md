## Getting Started

Follow these steps to run the **FoodLink** mobile app on your machine and launch it in the iPhone simulator.

### Requirements

Make sure you have the following installed:

- Node.js (v18 or newer recommended)

- npm

- Xcode (for iOS Simulator)

You can verify Node and npm with:

```bash
node -v
npm -v
```

### 1. Navigate to the Project

First, move into the application directory:

```bash
cd my-app
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start Expo

```bash
npx expo start -c
```

### 3. Run on the iPhone Simulator

After Expo starts in the terminal:

* Press i to open the app in the iOS Simulator
* Ensure Xcode is installed so the simulator is available

If the simulator doesn’t open automatically, open it from Xcode and press i again.
